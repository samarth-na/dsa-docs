# Array DSA Learning Roadmap

This roadmap provides a structured path to master array manipulation and problem-solving techniques. Follow this guide systematically to build a strong foundation in array-based Data Structures and Algorithms.

---

## Phase 1: Foundation Basics (Week 1-2)

### Objective

Build confidence with basic array operations and understand fundamental concepts.

### Daily Practice (5-7 problems per day)

#### Day 1-2: Basic Traversal & Manipulation

- `Reverse String` (344) — Easy — Two Pointers
- `Plus One` (66) — Easy — Math
- `Rotate Array` (189) — Medium — Two Pointers
- `Move Zeroes` (283) — Easy — Two Pointers

#### Day 3-4: Searching Fundamentals

- `Binary Search` (704) — Easy — Binary Search
- `Two Sum` (1) — Easy — Hash Map, Two Pointers
- `Two Sum II` (167) — Easy — Two Pointers, Binary Search
- `Find Pivot Index` (724) — Easy — Prefix Sum

#### Day 5-7: String-Array Hybrid Problems

- `Valid Palindrome` (125) — Easy — Two Pointers
- `Length of Last Word` (58) — Easy — String
- `Reverse Words in a String` (151) — Medium — String Manipulation
- `Reverse Words in a String III` (557) — Easy — String Manipulation

### Key Takeaways from Phase 1

- Master the two-pointer technique (start-end, slow-fast)
- Understand when to use Hash Maps vs Two Pointers
- Practice prefix sum calculations

## Tips for Success

1. **Understand, Don't Memorize** — Focus on understanding the approach rather than memorizing solutions.

2. **Time Your Practice** — Track how long it takes to solve problems. Target: Easy under 10 mins, Medium under 20 mins.

3. **Read Editorials** — Always check official solutions after solving, even if you got the answer.

4. **Practice Variations** — After solving a problem, think about how you could modify it.

5. **Build Your Notes** — Maintain a personal cheatsheet of patterns and tricky code snippets.
