## Quick Reference by Topic

### Arrays
- Linear scan for max/min/sum
- Prefix sum for range queries
- Two pointers for sorted arrays
- In-place modification patterns

### Strings
- Two pointers for palindrome/reverse
- Character counting with hash maps
- Word-based parsing

### Patterns
- Outer loop = rows, inner loop = columns
- Trace row-by-row before coding
- Identify symmetry vs. space-driven

### Number Theory
- `% 10` and `/ 10` for digits
- `i * i <= n` for prime checking
- Euclidean algorithm for GCD

### Matrix
- Boundary traversal for spiral
- Direction changes at edges
- Layer-by-layer approach

### Binary Search
- Sorted array prerequisite
- Check mid vs target
- Adjust left/right boundaries
