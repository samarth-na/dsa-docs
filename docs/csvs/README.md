# CSV Datasets

This section contains machine-readable datasets used by the docs.

## Available Files

| File | Purpose |
|---|---|
| `question_catalog.csv` | Master list of questions with description, topics, and difficulty |
| `patterns_questions.csv` | Detailed pattern-question metadata including sample input/output |
| `leetcode_questions.csv` | Compact LeetCode ID and title list |
| `question_topic_matrix.csv` | One-hot topic matrix for filtering, mapping, and analytics |

## Typical Use Cases

- Build filtered question lists outside the docs UI.
- Power visualizations like topic graphs in another repo/app.
- Track topic coverage and practice distribution.
- Export sheets for revision planning.
