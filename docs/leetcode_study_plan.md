# LeetCode Study Plan

This study plan arranges the repo's LeetCode questions by topic and learning order.
The order starts with easier pattern-recognition problems and then moves toward more complex ones.

## Stage 1: Warm-Up LeetCode Problems

Do these first to build confidence:
- `1` Two Sum
- `66` Plus One
- `704` Binary Search
- `724` Find Pivot Index
- `1991` Find the Middle Index in Array
- `896` Monotonic Array
- `9` Palindrome Number
- `58` Length of Last Word
- `3110` Score of a String
- `1678` Goal Parser Interpretation

Why first:
- Short implementations
- Easy debugging
- Builds comfort with arrays, loops, strings, and edge cases

## Stage 2: Arrays and In-Place Editing

Recommended order:
1. `26` Remove Duplicates from Sorted Array
2. `27` Remove Element
3. `283` Move Zeroes
4. `88` Merge Sorted Array
5. `169` Majority Element
6. `2529` Maximum Count of Positive Integer and Negative Integer
7. `189` Rotate Array
8. `238` Product of Array Except Self
9. `53` Maximum Subarray

What you learn:
- In-place overwrite logic
- Multiple scans versus single scan
- Keeping state while traversing
- Prefix and suffix style thinking

## Stage 3: Two Pointers

Recommended order:
1. `344` Reverse String
2. `125` Valid Palindrome
3. `167` Two Sum II - Input Array Is Sorted
4. `557` Reverse Words in a String III
5. `151` Reverse Words in a String
6. `15` 3Sum
7. `80` Remove Duplicates from Sorted Array II
8. `75` Sort Colors
9. `42` Trapping Rain Water

What you learn:
- Left-right movement
- Shrinking search space
- Duplicate handling
- When sorted input helps

## Stage 4: Math and Number Theory

Recommended order:
1. `202` Happy Number
2. `3658` GCD of Odd and Even Sums
3. `1492` The Kth Factor of N
4. `7` Reverse Integer

What you learn:
- Digit extraction
- Repeated transformation loops
- Factor counting
- Overflow awareness

## Stage 5: Strings and Hashing

Recommended order:
1. `389` Find the Difference
2. `242` Valid Anagram
3. `58` Length of Last Word
4. `3110` Score of a String
5. `1678` Goal Parser Interpretation
6. `151` Reverse Words in a String
7. `557` Reverse Words in a String III
8. `125` Valid Palindrome

What you learn:
- Character counting
- Parsing
- Cleaning spaces and punctuation
- Word-based traversal

## Stage 6: Matrix Problems

Recommended order:
1. `1572` Matrix Diagonal Sum
2. `566` Reshape the Matrix
3. `54` Spiral Matrix
4. `59` Spiral Matrix II
5. `48` Rotate Image
6. `73` Set Matrix Zeroes
7. `74` Search a 2D Matrix
8. `498` Diagonal Traverse

What you learn:
- Row-column indexing
- Boundary traversal
- In-place updates
- Direction changes and layer logic

## Stage 7: Greedy and Harder Array Problems

Recommended order:
1. `134` Gas Station
2. `31` Next Permutation
3. `877` Stone Game
4. `4` Median of Two Sorted Arrays
5. `42` Trapping Rain Water

What you learn:
- Start-point reasoning
- Lexicographic next state generation
- Strategic observations
- Advanced partition/search ideas
