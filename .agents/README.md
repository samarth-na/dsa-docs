# Agent Kit

Use these files as project-aware context for coding agents working in this repo.

- `project-context.md` - shared architecture, commands, and conventions.
- `opencode.md` - extra instructions for OpenCode-style CLI agents with tools.
- `gpt.md` - extra instructions for GPT models in chat or editor workflows.

Recommended usage:

1. Load `project-context.md` first.
2. Load either `opencode.md` or `gpt.md` based on the model.
3. Prefer these files over the root `agents.md` if they conflict; that file describes an older project setup.
